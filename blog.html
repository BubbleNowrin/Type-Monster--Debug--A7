The differences between Local Storage and Session Storage are: data stored in local storage have no expiration. On the other hand, data stored in session storage expires after the user switches the tab or close the browser.If we want to store data in client-side then local storage is used and if we store data in server side we use session storage.

The scope of a variable defines from where we can access it in a program based on where it was created. Global scope contains all of the things that are defined outside of all code blocks. A global variable has global scope. A global variable can be accessed from anywhere in the code. On the other hand, Block scope contains things that are defined inside code blocks. A local variable is only accessible where it is declared.

The JS event loop monitors the call stack and call back queue. This is the one task that the event loop handles. At first, it executes all the tasks in the call stack. If the call stack is empty,then it will take the first task from the callback queue and then push it to the call stack and eventually executes it.
